# Healthcare Facility Access Gap Analysis

A geospatial analytics platform that identifies underserved Nigerian communities using travel time modeling, facility density, healthcare workforce metrics, and K-Means gap-zone clustering. Guides evidence-based placement of new clinics and mobile health units.

---

## Features

- **Access Tier Scoring** — composite coverage score across 8 dimensions
- **Gap Zone Clustering** — K-Means identifies priority zones by affected population
- **State-Level Heatmaps** — comparative analysis across states
- **Interactive Streamlit Dashboard** — 4-page app with Plotly Mapbox maps

---

## Tech Stack

`Python` · `scikit-learn` · `Streamlit` · `Plotly` · `Pandas` · `NumPy`

---

## Setup & Run

```bash
git clone https://github.com/Momahmoses/healthcare-access-gap.git
cd healthcare-access-gap
pip install -r requirements.txt
streamlit run streamlit_app.py
```
